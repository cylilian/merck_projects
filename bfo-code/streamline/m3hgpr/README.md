config.py contains all available configurations. Important ones are: 
Model_name: specify model type 
Cate_trans: specify ways to convert categorical variables 
X_scale: specify ways to scale input data 
Y_scale: specify ways to scale output data 
N_CV: number of groups of data in cross-validation 
Output_rank, Task_rank, and Lik_rank are all Gaussian Process Model specific parameters
