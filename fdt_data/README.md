# Final Data All Excipients — Fast Disintegrating Tablet Dataset

## Description
This dataset provides detailed information on the formulation and physical characteristics of **Fast Disintegrating Tablets (FDTs)**, including:

- Active pharmaceutical ingredients (APIs)
- A wide range of excipients (binders, disintegrants, lubricants, sweeteners)
- Physicochemical properties (e.g., molecular weight, solubility)
- Tablet characteristics (e.g., disintegration time, hardness, friability)

The dataset contains **1982 formulations** and over **80 variables** extracted from peer-reviewed publications.

## Format
- File: `final_data_all_excipients.csv`
- Rows: 1982 formulations
- Format: CSV (machine-readable)

## Column Descriptions
A full data dictionary is provided in `data_dictionary.csv`, including:
- Variable names
- Descriptions
- Units
- Context (e.g., usage in formulation)

## Use Cases
- Machine learning model training for formulation prediction
- Optimization of excipient combinations
- Pharmaceutical data mining
- Teaching and research in pharmaceutics and drug delivery
